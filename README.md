# Portfolio

My first developer portfolio website. I've tried to keep it clean, mobile-first and taken inspiration from GitHub's profile aesthetic.

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Library**: React 19
- **Styling**: Tailwind CSS v4 (Class-based dark mode selector)
- **Package Manager**: pnpm

## 🚀 Getting Started

### Prerequisites

Ensure you have Node.js and `pnpm` installed.

### Installation

Clone the repository and install dependencies:

```bash
pnpm install
```

### Run Locally

Start the local development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to view your site.

### Build and Lint

To compile the production build:

```bash
pnpm build
```

To run the ESLint checker:

```bash
pnpm lint
```
